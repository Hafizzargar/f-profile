# Hafiz Basharat | Premium Full Stack Portfolio

A state-of-the-art, immersive portfolio website built with **Next.js**, **Three.js**, and **Framer Motion**. This project showcases high-performance UI components, interactive 3D visualizations, and a sophisticated design system.

## 🚀 Live Demo
The project is currently being deployed on Vercel. 

## ✨ Key Features

- **Interactive 3D Hero**: Features a custom-animated, reactive 3D sphere built with React Three Fiber (Three.js) that responds to user presence.
- **Dynamic Spotlight Effect**: A mouse-following radial spotlight that adds depth and interactivity to the dark-mode background.
- **Premium Glassmorphism UI**: High-end card designs with backdrop filters, custom noise textures, and smooth scaling transitions.
- **Technical Excellence Grid**: Categorized view of skills including Frontend Frameworks (Angular, React, Next.js), UI libraries (PrimeNG, Tailwind), and Backend (Node.js, MongoDB).
- **Professional Timeline**: A stylized vertical journey tracking experience at Wealthmax and AccioJob.
- **Responsive Architecture**: Fully optimized for all device sizes with a mobile-first approach and fluid typography.

## 🛠️ Tech Stack

- **Framework**: [Next.js 16](https://nextjs.org/) (App Router & Turbopack)
- **Frontend**: [React 19](https://reactjs.org/), [TypeScript](https://www.typescriptlang.org/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **3D Engine**: [Three.js](https://threejs.org/) via [@react-three/fiber](https://github.com/pmndrs/react-three-fiber)
- **Styling**: Vanilla CSS (Global Variables & Utility Classes)
- **Icons**: [Lucide React](https://lucide.dev/)

## 🛠️ Getting Started

### Prerequisites
- Node.js 20+ 
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Hafizzargar/f-profile.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## 🚢 Deployment

This project is optimized for [Vercel](https://vercel.com). To deploy:
1. Push your code to GitHub.
2. Import the repository into Vercel.
3. Select **Next.js** as the framework preset.
4. Deploy!
